### Description of the system

This web application is meant to find the nearest libraries to a specific school. We remembered our need to know where we could get needed literature or to simply know a nearby quiet library where we could study more effectively. For that purpose, we decided to make it easier for the other students to get access to the closest libraries near their school. It will allow them to choose their school and the system will provide them with the needed information.

### Functional requirements

The web application shall be connected with a database of schools and libraries in Skopje.

The web application shall have a selecting function for the provided list of schools in Skopje.

The web application shall display a map with the libraries in Skopje and their info.

The web application shall have a calculating function for distance between a school and the libraries.

The web application shall display the nearest libraries to the selected school.

### Non-functional requirements

The web application shall be able to run on different operating systems (Windows, MacOS, Linux) and support most browsers such as Chrome, Firefox, Safari etc.

The web application shall respect privacy policy.

The calculating function shall not take more than 10 seconds.

The precision of the calculating function shall be with an accuracy of 98%.

The web application shall enable novice users to perform the select function in less than 1 minute.
